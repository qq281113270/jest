如何将md文件转换成带目录的html文件

配置环境node

  • 去官网下一个node安装包，下一步下一步：

    由于现在的node都自带npm，

    直接

    npm install -g i5ting_toc

这样安装好了i5ting_toc这个包，

进入你实现准备好md文件，然后通过命令行进入到该文件

i5ting_toc -f note.md

这里note.md是你自己的文件名

修改最外层第一个标题

运行上面命令之后会生成一个preview文件夹，文件夹有js，style，js和html，html就是会生成带有目录的html 并且带跳转。但是外层的标题名字默认为"Table of Content" 那么要修改他，就要把html的底部的js修改为一下内容

$(document).ready(function(){
    var css_conf = eval(markdown_panel_style);
    $('#readme').css(css_conf)
    var conf = eval(jquery_ztree_toc_opts);
    conf={
        ...conf,
        _header_nodes: [{ id:1, pId:0, name:"将md文档转换成html并且带标题",open:true}],
    }
        $('#tree').ztree_toc(conf);
});

主要是修改_header_nodes里面的name即可
